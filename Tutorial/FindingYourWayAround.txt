Welcome
Welcome
Distant Worlds のチュートリアルへようこそ。このチュートリアルでは、あなたの帝国を実行する方法の基礎を学びます。

このチュートリアルの各手順を進行する [Continue] ボタンをクリックしてください。

あなたは、マウスでドラッグして、このチュートリアルのウィンドウを移動できます。

始めましょう：下の [Continue] ボタンをクリックしてください。
~
Introduction
Introduction
Distant Worlds は複雑なゲームですが、目標は単純です：銀河帝国の構築と維持。

ゲームでは、4つの主要分野に焦点を当てています：
- 銀河を探索
- 新惑星を植民地化
- 価値ある資源を抽出
- あなたの帝国を防衛
~
The Galaxy
The Galaxy
銀河は大きい場所です - 求める新世界、発見する異人種、搾取する資源が 満載です。

銀河の周りを移動する方法を見てみましょう。
~
Moving Around
Moving Around
メインビューは、銀河の一部を示しています -この場合、星系の一部は、あなたのコロニー惑星の一つの近くを示しています。

上下左右にメインビューをスクロールするには、画面の端にマウスポインタを移動します。同じことを行うには、キーボードの [ 矢印 ] キーを使用することができます。

また、マウスの右ボタンを押したままにして、メインビューをドラッグすることができます。
~
Moving Around - Zooming
Moving Around - Zooming
銀河の大小を見るために拡大/縮小で、メインビューをズームすることができます。

拡大/縮小でメインビューをズームするには、マウスのスクロールホイールを使用してください。同じことをするために [PageUp] キー と [PageDown] キー を使用することができます。
~
Zoom buttons
Zoom buttons
メイン画面上の [ZoomIn] ボタンでと [ZoomOut] ボタンでも、拡大/縮小することができます。
~
System view
System view
星系レベルまでズームアウトするときは、帝国のメインカラーによって帝国のコロニーや艦船を識別することができます。

あなたが [Ctrl] キーを押したままクリックすると、任意の場所を拡大することができます。
~
Sector view
Sector view
さらにズームアウトして、メインビューの星系表示を更新します。

このレベルでは、艦船や拠点がアイコンとして表示されます。アイコンの色は艦船や基地が属する帝国を示しています。
~
Explored Systems
Explored Systems
探索された星系の隣には、それらの名前が表示されています。
~
Unexplored Systems
Unexplored Systems
探索されてない星系の隣には、それらの名前が表示されていません。
~
Colonized Systems
Colonized Systems
帝国によって植民地化された星系は、その帝国の色に囲まれています。

円の大きさは、星系の相対的な重要性を示しています - 大規模星系は、より貴重です。
~
Empire Territory
Empire Territory
帝国領土の一部の星系では、着色された背景を持っています。

それが植民地化されていない場合でも、星系を帝国が所有することができます。

コロニーは、近くの星系の帝国の影響力によって、彼らの帝国の領土の一部として示されます。
~
Independent Systems
Independent Systems
一部のコロニーでは、任意の帝国に属していない宇宙人の独立したコロニーが存在します。

これらの独立した星系は、グレーの実線で囲っています（チュートリアル中、存在しない場合が多い）

彼らは、植民地化するために、あなたの帝国の良い標的にすることができます - 異星人は、あなたの帝国のためにすぐさま収入を得て開始することができ、あなたの新しいコロニーの住民に重大な向上を提供します。
~
Systems with Potential Colonies
Systems with Potential Colonies
潜在的なコロニーを持つすべての星系は、灰色の破線の円で示されます（チュートリアル中、存在しない場合がままある）｡

新しいコロニー艦を構築し、自分の帝国のための新しい植民地を確立するため、これらの無人の惑星にそれを送ってください。
~
Zoom presets
Zoom presets
[ズーム] ボタンを使用すると、即座にいくつかの決められたレベルにズームすることができます：
- 100% (default)
- 星系ビュー (system view)
- セクタービュー(sector view)
- 銀河ビュー (galaxy view)
~
Galactopedia Help
Galactopedia Help
Galactopedia は、現在選択されている項目や、現在開いている画面上に状況依存ヘルプを提供します。

それは、惑星とあなたが銀河の中で遭遇する艦船のすべての異なるタイプに関する詳細な情報が含まれています。また、ゲームの概念を説明し、各ゲーム画面について説明します。

[F1] キーを押すか、[ヘルプ] ボタンをクリックすれば、いつでも Galactopedia ヘルプシステムにアクセスすることができます。
~
Main Screen elements
Main Screen elements
メイン画面上で他の要素のいくつかを、ざっと見てみましょう。
~
Selection Panel
Selection Panel
画面左下の選択パネルです。

これは、植民地、艦船、星系等であるかどうか、選択した項目の詳細情報が表示されます

メイン画面で項目を選択するには、単にマウスでクリックします。
~
Empire Navigation Tool
Empire Navigation Tool
画面の左側にある帝国ナビゲーションツールです。これはあなたの帝国内のすべてへの迅速なアクセスを提供する、スクロール可能なリストの集まりです。

このような項目が含まれています：あなたのコロニー、宇宙港、建設艦、探査艦、敵目標、艦隊、新しい潜在的なコロニーは、潜在的な採掘場所、さらにもっと。

リストのいずれかを開くには、アイコンをクリックします。リストを閉じるには、リストのタイトルバーの右にある 閉じる ボタンを使用するか、もう一度アイコンをクリックします。
~
Empire Navigation Tool contd.
Empire Navigation Tool contd.
下記の選択パネルでそれを選択するには、リスト内の項目をクリックします。

選択項目の多くの一般的な任務にアクセスできる選択パネルの下の [ アクション ] ボタンは、簡単にメイン画面から直接あなたの帝国の大部分を管理することができます。

リスト内の項目をダブルクリックすると、その項目にメインビューを移動します。

セクターや銀河レベルにズームアウトさせた時、そのリスト内の項目上に移動すると、拡大した黄色い丸で銀河内のその項目の位置を伝えます。
~
Map
Map
右下は、マップです。これは、周辺地域を示しています。

メインビューに表示されている領域は、マップの中央の青い四角形で示されます。

その場所に、メインビューを移動するには、地図上の任意の場所をクリックしてください。

メインビューをズームすると、マップは自動で拡大/縮小します。
~
Scrolling message list
Scrolling message list
画面の上部にあなたの帝国で発生した最近のイベントが表示されたリストです。

イベントの場所に移動するには、このリスト内のイベントをクリックします。
~
Pause/Resume
Pause/Resume
画面の左上に [一時停止] ボタンがあります。これにより、いつでもゲームの一時停止や再開できます。

ゲームを一時停止するには、一回クリックします。ゲームを再開するには、もう一度クリックします。
~
Game Menu
Game Menu
画面の左上に [ゲームメニュー] ボタンがあります。

ゲームの保存やロード、ゲームのオプション変更、ゲーム終了をする場合、ゲームメニューにアクセスします。
~
Game Options
Game Options
ゲームのオプション画面（[O] キー）を使用すると、ゲームの設定をカスタマイズすることができます。

具体的には、植民地化や造船などのような特定の帝国のタスクを自動化することができます。これは、あなたの興味をもっと帝国の他の領域に集中することができます。

あなたが最初に Distant Worlds を起動すると、多くの任務は、あなたのために自動化されています。あなたの帝国の詳細を細かく制御望むならば、これらの任務の自動化は徐々に、オフにすることができます。
~

Completed

準備ができたら、あなたはまた、メインメニューからほかのチュートリアルを試すことができます。
